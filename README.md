# 🌱 Digital Life Lessons

Digital Life Lessons is a full-stack web application where users can create, organize, and share meaningful life lessons. Users can explore public lessons, save favorites, interact through likes and comments, and upgrade to Premium to unlock exclusive content.

---

## 🎯 Project Purpose

The purpose of this project is to help users preserve valuable life experiences, share personal growth insights with others, and build a community where people can learn from real-life lessons.

---

## ✨ Key Features

- 🔐 Better Auth authentication (Email/Password & Google Login)
- 👤 User and Admin dashboards
- 📝 Create, update, and delete life lessons
- 🌍 Public and Private lesson visibility
- 💎 Free and Premium lesson access
- 💳 Stripe Lifetime Premium subscription
- ❤️ Like and save favorite lessons
- 💬 Comment on lessons
- 🚩 Report inappropriate lessons
- 🔍 Search, filter, sort, and pagination
- 📊 Dashboard analytics and statistics
- 📱 Fully responsive design
